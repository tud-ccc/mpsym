Detailed Description

Data structure for space efficient storage of orbit transversals.

The orbit of a group element \(g \in G\) where \(G\) acts on the set \(\Omega\) is the set \(G(x) = \{x^g \mid g \in G\}\). The orbit transversals corresponding to this orbit are \(\{u_{\beta} \mid \beta \in G(x)\}\) where \(u_{\beta} \in G\) maps \(x\) to \(\beta\). Instead of storing these explicitly (which can might consume a lot of space for large orbits (possible for large sets \(\Omega\)), this data structure only stores a spanning tree for the orbit graph connecting every element \(\beta' \in G(x)\) to every other element \(\beta\) via a directed edge labelled with \(g \in G(x)\) if \(\exists g \in G \colon \beta^g = \beta'\). The spanning tree is rooted at \(x\).
